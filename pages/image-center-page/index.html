自动生成标签流程配置

⚙️ 自动标签生成设定

设置用于图库图片自动特征标签生成的图片转述模型。

默认继承 AstrBot 系统活动配置文件中的“默认图片转述模型”设置；保存后会同步写入系统配置。

需要选择一个支持图片理解的可用图片转述模型。

✍️ 生成标签类别设定

控制插件为图片自动赋予特征标签时，优先基于哪些类别来生成。

可勾选的预设类别：图像类别 (是照片还是表情包)、人物特征、身材、情绪、动作、图片中的文本。

可输入多个自定义类别，各个类别之间可以用空格、逗号或顿号隔开。

注意：该操作会清空图库中原先的特征标签 (公用标签除外)

Proactive Emoji Reply

对话中主动发送表情包

📤️ 对话中主动发送表情包

调用 LLM 对 bot 回复用户的内容进行语义分析，从图库中选择并发送符合语境的表情包。

当 bot 的对话由 LLM 生成，且概率命中时，启动语义情感分析 + 表情包发送流程。

继承 AstrBot 系统配置中的模型提供商；留空或不可用时，继承当前会话的模型选择策略。建议选择实时性更强的多模态模型，如 gpt-5.4-mini 或 mimo-v2.5。

串行模式依据 bot 生成回复内容进行表情包检索，语境分析更准确；并发模式会同步进行回复生成+表情包检索，减少等待时间；本地精排+并发模式会先按标签库和强语义命中精排候选，速度最快。

注意：非 GPT 模型可能因 API 请求频率限制，导致串行检索速度偏慢。建议使用并发检索模式。

开启后，只从带“表情包”特征标签的图库图像中筛选。

开启后，表情包和对话在同一次对话中一起发送；关闭后，对话遵循原先的 LLM 普通对话发送流程，表情包会以非 LLM 输出格式独立发送。关闭此选项可避免与其他插件产生兼容性冲突。

取值范围 0-1。只有普通 LLM 回复概率命中时，才启动语义和情感分析 + 表情包选择流程。

AI 分析/选择表情包的最长等待时间，超时后跳过主动表情包追加。

仅对主动表情包流程开启详细运行日志，用于排查 LLM 查询、接口调用和模型 API 报错状态。

开启后，主动发送表情包成功时，将标签追加到本轮 assistant 回复内容中；不会把标签注入下一轮用户消息。

Auto Image Collection

自动收集群友发的图

📩 自动收集群友发的图

在指定 QQ 群中收集图片。图片先进入 [待筛选图片池]，用户需要选择哪些图像加入 [固化图像库]，入库后才会自动打标签、被检索。

开启后，插件会在指定 QQ 群中自动收集图片，放入 [待筛选图片池]。

(建议关闭) 关闭时，用户可手动指定 [待筛选图片池] 中哪些图片存入 [固化图像库]。开启后，图像将实时自动入库。

开启后，[固化图像库] 中的图片，会和手动上传图像一起，作为“为用户寻找表情包”和“对话中主动发送表情包”的检索候选项。

每行一个 QQ 群号。机器人只在这些群聊中自动收集图片。

控制自动收集时如何处理照片、截图等图像。

每行一个 QQ 号。插件不会收集他们在任意群聊中发送的图片。

开启后，会记录被 [待筛选图片池] 手动丢弃的图片信息；再次见到相同图片时直接跳过。

设置为 -1 表示无上限。

Group Meme Combat

群聊智能斗图

🥊 群聊智能斗图

在群聊图片对话中自动加入队形、按概率连发相关图片，或在连续图片团战中检索语义相近的图库图片。

功能总开关。关闭后，不会统计群聊图片窗口，也不会触发加入队形、图片连发或参与团战 (即使打开了下面的子功能开关)。

加入队形

如果多名群友在时间窗口内连续发送相同图片，bot 会跟着队形进行接龙。

注意：请设置为 >=2，以避免 bot 跟着单个用户发同一张图。

开启后，相同用户在统计窗口内多次发送同一张图片只计为一次；必须达到指定数量的不同用户发送同一张图片，bot 才会加入队形。

图片连发

bot 有一定概率在某次图片回复中，连续发送多张表情包。

取值范围 0-1。其他插件控制 bot 发送的图片不计算。

bot 最开始发出的那张图片不计入连发数量。

参与团战

如果群聊进入斗图团战 (对话中连续出现图片)，bot 会抽取其中 2 张图片快速分析语义，从自己的图库中搜索表情包参与团战。

留空或不可用时继承 AstrBot 当前会话模型选择策略。建议选择实时性更强的多模态模型，如 gpt-5.4-mini 或 mimo-v2.5。

Qwen 模型的图片转述速度可能较慢。若有实时性需求，请使用其他模型。

User Image Search

为用户寻找表情包

🔍️ 寻找表情包功能开关

用户唤醒机器人后，如果对话中存在下列关键词，则在图库中检索满足要求的表情包。

关闭后，即使用户对话包含下方关键词，也不会触发机器人为用户检索并发送图库图片；不会影响“对话中主动发送表情包”功能。

用户唤醒机器人后，消息中包含任一关键词才会进入本插件流程。注意：自动标签生成必须在 AstrBot 的 [系统配置] 中，配置 [默认图片转述模型]。

支持 {filename}、{tags}、{need}、{reason} 占位符。
